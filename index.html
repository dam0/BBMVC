Welcome to BBMVC.

An extreamly light MVC framework that is written in Haxe that you can use for your Actionscript, and other cross platform Haxe projects.

Easy install.

If you have haxelib installed then open up a terminal window and type $> haxelib install bbmvc . Now you can add it to your hxml file -lib bbmvc and you should be good to go.

Actionscript Example

If you would like to use BBMVC in your Flex or Actionscript projects it's pretty simple. I've provided an Actionscript example to help you get started.

IMPORTANT After you have grabbed the source code for BBMVC github page you will want to grab the bbmvc.swc out of the build directory and drop it into your Actionscript project in order for it to work.
